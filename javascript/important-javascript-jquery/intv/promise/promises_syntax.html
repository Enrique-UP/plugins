"Promises are used to handle asynchronous operations like API calls, file reading, or database requests. They prevent callback hell and make asynchronous code easier to read and maintain. A Promise has three states: Pending, Fulfilled, and Rejected. We handle success using .then() and errors using .catch(). Most modern APIs, including fetch(), return Promises."

    const promise = new Promise(function(resolve,reject){
        let success = true;
        if(success){
            resolve("Data Found");
        }else{
            reject("Server Error");
        }
    });
