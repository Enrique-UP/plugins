Math Functions

  1. attr() Returns the value of an attribute of the selected element
  2. calc() Allows you to perform calculations to determine CSS property values
  3. conic-gradient() Creates a conic gradient
  4. counter() Returns the current value of the named counter
  5. cubic-bezier() Defines a Cubic Bezier curve
  6. hsl() Defines colors using the Hue-Saturation-Lightness model (HSL)
  7. hsla() Defines colors using the Hue-Saturation-Lightness-Alpha model (HSLA)
  8. linear-gradient() Creates a linear gradient
  9. max() Uses the largest value, from a comma-separated list of values, as the property value
  10. min() Uses the smallest value, from a comma-separated list of values, as the property value
  11. radial-gradient() Creates a radial gradient
  12. repeating-conic-gradient() Repeats a conic gradient
  13. repeating-linear-gradient() Repeats a linear gradient
  14. repeating-radial-gradient() Repeats a radial gradient
  15. rgb() Defines colors using the Red-Green-Blue model (RGB)
  16. rgba() Defines colors using the Red-Green-Blue-Alpha model (RGBA)
  17. var() Inserts the value of a custom property
