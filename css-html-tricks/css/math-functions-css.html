Math Functions

  1. var() Inserts the value of a custom property
  2. attr() Returns the value of an attribute of the selected element
  3. calc() Allows you to perform calculations to determine CSS property values
  4. min() Uses the smallest value, from a comma-separated list of values, as the property value
  5. max() Uses the largest value, from a comma-separated list of values, as the property value
  6. conic-gradient() Creates a conic gradient
  7. counter() Returns the current value of the named counter
  8. cubic-bezier() Defines a Cubic Bezier curve
  9. hsl() Defines colors using the Hue-Saturation-Lightness model (HSL)
  10. hsla() Defines colors using the Hue-Saturation-Lightness-Alpha model (HSLA)
  11. linear-gradient() Creates a linear gradient
  12. radial-gradient() Creates a radial gradient
  13. repeating-conic-gradient() Repeats a conic gradient
  14. repeating-linear-gradient() Repeats a linear gradient
  15. repeating-radial-gradient() Repeats a radial gradient
  16. rgb() Defines colors using the Red-Green-Blue model (RGB)
  17. rgba() Defines colors using the Red-Green-Blue-Alpha model (RGBA)
