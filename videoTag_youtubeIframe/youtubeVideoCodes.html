Autoplay + Muted
For Google Chrome
chrome://settings/content/sound => Default behaviour => Sites can play sound (select this option)

For Firefox
about:preferences#privacy => Default for all websites: Allow Audio and Video
mute=0 (mute)
mute=1 (unmute)

Loop
Add loop=1 to let your video loop forever.
Value 0 (default): The video will play only once.
Value 1: The video will loop (forever).
