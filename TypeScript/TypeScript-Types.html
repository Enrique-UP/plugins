TypeScript Types Ki Ultimate Cheat-Sheet 🚀

Category Type Name Kaam Kya Hai? (Description) Code Example
Core Primitives (Basic & Simple)
:string Text ya words ko store karne ke liye. const name: string = "Brave";
:number Har tarah ke numbers (integers aur decimals) ke liye. const age: number = 26;
:boolean Sirf haan ya naa (true ya false) ke liye. const isCoder: boolean = true;
:null Intentional khali value (jaan-bujhkar empty rakhna). const car: null = null;
:undefined Uninitialized value (jiski value abhi tak decide nahi hui). let x: undefined = undefined;
:bigint Bahut bade integers ke liye (jo normal number se bade hon). const bigNum: bigint = 9007199n;
:symbol Ek unique identity/key banane ke liye. const sym: symbol = Symbol("id");
TypeScript Special (TS Exclusives)
:any Kuch bhi chalega (Type safety ko band kar deta hai - Avoid it). let data: any = "hello"; data = 12;
:unknown Type-safe any. Pehle check karo ki isme kya hai, tabhi use hoga. let val: unknown = "Enrique";
:void Jab koi function kuch bhi return nahi karta. function log(): void { console.log("Hi"); }
:never Jo kaam kabhi khatam hi nahi hota (jaise infinite loop ya errors). function err(): never { throw "error"; }
Complex Types (Structure & Groups)
Type[] / Array[Type] Ek hi type ke multiple items ki list ke liye. 
const marks: number[] = [90, 85, 70];

const students: Array<string> = ["Amit", "Rahul", "Priya"];

const students: string[] = ["Amit", "Rahul", "Priya"];

const attendance: boolean[] = [true, false, true, true];

const attendance: Array<boolean> = [true, false, true, true];

const mixedData: (string | number)[] = ["Amit", 90, "Priya", 85];

const mixedData: Array<string | number> = ["Amit", 90, "Priya", 85];


const classA: Student[] = [
    { name: "Amit", rollNo: 101 },
    { name: "Priya", rollNo: 102 }
];



const classA: Array<Student> = [
    { name: "Amit", rollNo: 101 },
    { name: "Priya", rollNo: 102 }

[Type, Type] (Tuple) Fixed length aur fixed positions wale array ke liye. const coordinate: [number, number] = [12, 45];
const response: [number, string] = [404, "Not Found"];
:object Kisi bhi complex key-value structure ke liye. const user: object = { name: "Enrique" };
enum Agar aap koi value assign nahi karte hain, to TypeScript automatic 0 se shuru karke number assign kar deta hai: 
                        
enum CardinalDirections {
  North,
  East,
  South,
  West
};
let currentDirection = CardinalDirections.North;
// North is the first value so it logs '0'
console.log(currentDirection); // 0
                        
                        
enum CardinalDirections {
  North = 1,
  East,
  South,
  West
};

// logs 1 since we initialized the first value to something other than '0'
console.log(CardinalDirections.North); // 1

// logs 4 as it continued on from the initial '1' value
console.log(CardinalDirections.West); // 4                            
                        
                        
enum StatusCodes {
  NotFound = 404,
  Success = 200,
  Accepted = 202,
  BadRequest = 400
};

// logs 404
console.log(StatusCodes.NotFound); // 404

// logs 200
console.log(StatusCodes.Success); // 200                            
                        
                        
enum CardinalDirections {
  North = 'North',
  East = "East",
  South = "South",
  West = "West"
};

// logs "North"
console.log(CardinalDirections.North);

// logs "West"
console.log(CardinalDirections.West);
